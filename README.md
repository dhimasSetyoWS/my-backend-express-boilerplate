# Foobar

Foobar is a Python library for dealing with word pluralization.

## Installation

Use the npm (node package manager)

```bash
npm install
```

Setelah penginstallan, langkah selanjutnya menjalankan prisma

```bash
npx prisma init
```

Kemudian ubah DATABASE_URL pada env-example sesuai dengan URL database.

## Buat Skema

```
prisma > schema.prisma
```


## Usage

```bash
npx prisma migrate dev
```

Ini akan menjalankan migrate prisma sesuai dengan schema yang telah kita buat tadi
